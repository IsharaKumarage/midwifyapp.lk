Abstract

Bridging the Gap in Maternal Healthcare

Sri Lanka's midwifery care system faces a critical digital divide. Frontline midwives lack access to intelligent, real-time tools capable of detecting maternal risk factors, monitoring infant cranial development, interpreting complex fetal health signals, and receiving immersive practical training.

The Midwify platform comprises: a machine-learning-powered mobile app for real-time maternal risk prediction; a bilingual AI platform for infant cranial deformity and posture monitoring; an automated fetal health assessment system interpreting CTG data; and an immersive VR training environment — delivering Sri Lanka's first offline-capable, multilingual digital midwifery ecosystem.

🔬

Problem

Midwives lack real-time intelligent tools for detecting maternal and fetal risks across rural Sri Lanka.

💡

Novelty

First integrated, bilingual, offline-capable ML + VR midwifery platform built for South Asia.

🎯

Impact

Empowers 15,000+ Sri Lankan midwives with AI-driven decision support and immersive training.

Our Research

Four Integrated Components

Each component targets a distinct clinical need, together forming a complete digital midwifery ecosystem.

📱
Component 01

ML Maternal Risk Prediction

Real-time maternal risk classification for midwives using ensemble ML models deployed on a Flutter mobile app with offline TFLite inference.

🤖 Random Forest📊 XGBoost📱 Flutter🔥 Firebase⚡ TFLite
👶
Component 02

AI Infant Cranial Monitoring

Bilingual (Sinhala/Tamil) AI app detecting infant cranial deformity and postural imbalance using CNN-based computer vision analysis.

🧠 CNN👁️ Computer Vision🌐 Bilingual📱 React Native
🫀
Component 03

Fetal Health Risk Assessment

Automated CTG data interpretation classifying fetal health into Normal, Suspect, or Pathological states — reducing diagnostic error.

📈 CTG Analysis🤖 Random Forest👁️ Computer Vision📱 Flutter
🥽
Component 04

VR Training Simulation

Immersive Unity/OpenXR VR simulation for newborn handling training with Blender assets and interactive voice guidance.

🎮 Unity🥽 OpenXR🎨 Blender

Research Gap

Gaps in Existing Solutions

⚠️

No Integrated ML Risk Tools

Existing maternal health apps focus on single parameters. No platform combines multi-factor ML risk scoring for Sri Lankan midwives in an offline, bilingual format.

🔍

Limited Infant Monitoring

No accessible, AI-driven mobile tool exists for detecting infant cranial deformity or postural asymmetry in resource-limited clinical environments.

🥽

No Integrated VR + Postnatal Tools

No single platform combines immersive VR newborn-handling training with digital learning support for midwifery students in South Asia.

Objectives

Research Objectives

📱

Component 01 — Maternal Risk

  • Train Random Forest & XGBoost on clinical datasets
  • Deploy via TFLite on Flutter for offline use
  • Real-time dashboard with Firebase sync
👶

Component 02 — Infant Cranial AI

  • Build CNN model for CVAI and CI index assessment
  • Deliver bilingual Sinhala/Tamil React Native app
  • Offline-capable with on-device inference
🫀

Component 03 — Fetal CTG Assessment

  • Extract CTG parameters with OpenCV & PyTesseract
  • Classify using SMOTE-enhanced Random Forest
  • Mobile-compatible offline interface
🥽

Component 04 — VR Training

  • Develop Unity/OpenXR VR simulation environment
  • Create 3D assets using Blender for realism
  • Interactive newborn-handling training scenarios
